# Templates für Minecraft Bedrock Addons

Dieser Ordner enthält Templates für den Addon-Export.

## Dateien

### manifest_behavior.json
Template für Behavior Pack Manifest (Minecraft Bedrock 1.21.130+)

**Platzhalter:**
- `{{NAME}}` - Projekt-Name
- `{{DESCRIPTION}}` - Addon-Beschreibung
- `{{HEADER_UUID}}` - UUID für Header (wird automatisch generiert)
- `{{MODULE_UUID}}` - UUID für Module (wird automatisch generiert)
